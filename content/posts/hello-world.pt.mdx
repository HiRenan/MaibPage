Este é o primeiro post do blog. Ele existe por um motivo simples: o site é a prova.
Um lugar que fala de engenharia de IA tem que ser, ele mesmo, bem engenheirado.

## O que vive aqui

Notas técnicas sobre Claude Code, programação determinística de LLM e o que eu vou
descobrindo enquanto construo a MAIB. Sem enrolação, só o que eu de fato uso e o
que dá errado no caminho.

- meu ponto de vista sobre IA e o fluxo de dev
- decisões de arquitetura, com o porquê por trás
- experimentos, inclusive os que dão errado

> O que está na frente vem de executar, não de decorar. Cada risco que eu corro aqui é de propósito.

### Como o conteúdo é escrito

Cada post é um arquivo MDX no repositório, validado no build. O frontmatter passa
por um schema; se algo estiver errado, o build quebra de propósito. O indexador
vive em `lib/posts.ts` e é a fonte única da verdade:

```ts
import { getAllPosts } from '@/lib/posts';

const posts = getAllPosts('pt'); // ordenados por data, já validados
```

O pipeline também destaca o código com tema escuro e respeita a largura de leitura.
O código deste site é aberto no [repositório](https://github.com/HiRenan/MaibPage),
e você sempre pode voltar pra [home](/).
